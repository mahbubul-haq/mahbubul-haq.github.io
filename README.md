# Personal Portfolio Website

My personal portfolio website is built with React and Material UI. It showcases my projects, skills, experience, and contact information with modern styling, animations, and dark mode support.

## Features

- Responsive layout
- Dark mode
- Project cards with embedded content and image previews
- Skills, experience, and contact sections
- Smooth page animations and scroll behavior

## Technologies

- React
- Material UI

## How to run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

This project is configured to deploy using GitHub Pages.

```bash
npm install
npm run build
npm run deploy
```

The deploy command publishes the `dist` folder to the `gh-pages` branch using the `gh-pages` package.

## Notes

- The `homepage` field in `package.json` is set to `https://mahbubul-haq.github.io`.
- If you change the deployment URL, update the `homepage` value accordingly.
